## *✨ Features*  
- 🎨 *6 Dynamic Themes + Dark Mode*  
- 🚀 *GSAP & AOS Animations* (60 FPS optimized)  
- 📱 *Fully Responsive* (Mobile to 4K)  
- 📩 *EmailJS Contact Form* (Messages sent directly to your inbox) 

## *🛠 Tech Stack*  
- *Frontend*: React + Vite  
- *Animations*: GSAP, AOS  
- *Styling*: CSS3, Flexbox/Grid  
- *Deployment*: Vercel
